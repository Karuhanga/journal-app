# Project Title

Journal.ly is a little utility to keep you in sync with all your plans and thoughts.

## Getting Started

### For the user
Simply get a hold of the .apk file, install it on an android device, and pour your thoughts out.

### For the dev
Journal.ly supports android versions kitkat and <. Simply clone the repo, and follow the straight foward cooments given in each file. Cheers.

### Prerequisites

### For the user
Android Lollipop and greater.

### For the dev
Android Studio 3 and above should get you moving without any hiccups.
Gradle 4.1 and greater.
